# 💰 [vibers] Business Model & Cost Analysis (v1.0)

사용자가 스스로 가장 잘 쓰고 싶은 서비스를 만드는 것, 그것이 가장 강력한 성공 방정식입니다. 레오님이 첫 번째 '헤비 유저'가 되는 순간, 이 서비스의 완성도는 보장된 셈입니다.

---

## 1. 운영 비용 구조 (Cost Structure)

우리는 '거인들의 어깨 위에서 노는' 전략을 사용하여 고정비를 최소화합니다.

| 항목          | 플랫폼            | 비용           | 비고                                                |
| :------------ | :---------------- | :------------- | :-------------------------------------------------- |
| **코드 저장** | GitHub API        | **$0 (무료)**  | 사용자 개인 계정 사용으로 무제한 확장 가능          |
| **미리보기**  | Sandpack SDK      | **$0 (무료)**  | 사용자 로컬 브라우저 자원 활용                      |
| **실전 배포** | Vercel API        | **$0 (무료)**  | Vercel의 무료 플랜(Hobby) 활용                      |
| **생성 지능** | Gemini 3.0 API    | **변동비**     | **유일한 주요 비용.** 사용자의 대화량(Token)에 비례 |
| **DB/서버**   | Supabase/Firebase | **$0~ (낮음)** | 사용자 프로필 및 크레딧 관리용                      |

---

## 2. 사용자 등급 정책 (Tiers)

### 🥑 **Starter (Free)**

- **대상**: 호기심 많은 입문자 및 계발자들
- **제공**:
  - **Welcome 100 Vibe Credits** (가입 시 즉시 지급)
  - 이 크레딧으로 평균적으로 **3개 내외의 시제품**을 제작할 수 있습니다.
- **특징**:
  - 프로젝트 개수에 상관없이 크레딧 한도 내에서 자유롭게 사용 가능.
  - 한 프로젝트에 깊게 몰입할 경우 하나의 프로젝트에서 100크레딧이 모두 소모될 수 있습니다. (유연한 사용 권장)
  - 엔진: Gemini 3.0 Flash (빠른 반응 속도)

### 🥇 **Vibers Pro (Subscription)**

- **대상**: 나만의 브랜드를 직접 구축하고 운영하는 진정한 바이버스
- **혜택**:
  - 프로젝트 생성 **무제한**
  - 고급 엔진: Gemini 3.0 Pro/High (복잡한 로직 및 최상위 퀄리티)
  - **시크릿 레시피** 라이브러리 연동 (원터치 디자인 템플릿)
  - 전용 도메인 매핑 가이드 및 에이전트 서포트

---

## 3. 크레딧 시스템 (Vibe Credits)

- **사용 기반 차감**: 파일 생성, 복잡한 코드 수정, 반복적인 시뮬레이션 등 AI 자원을 많이 사용하는 행동에 따라 크레딧이 차감됩니다.
- **충전식**: "1,000 바이브 충전 (₩11,000)" 등 필요한 만큼만 구매하여 창작 활동을 이어갈 수 있습니다.
- **구독 포함**: Pro 구독 시 매월 넉넉한 양의 크레딧을 정기적으로 충전해 드립니다.
- **Expert Call (Human-in-the-Loop)**: AI가 해결하지 못하는 복잡한 문제 발생 시, 숙련된 '선배 바이버스'에게 크레딧을 지불하고 직접적인 코드 리뷰나 조언을 요청할 수 있습니다.

---

## 4. 커뮤니티 수익화 (Vibers Ecosystem)

우리는 단순히 툴을 제공하는 것을 넘어, 사용자 간에 지식이 거래되는 마켓플레이스를 지향합니다.

- **Vibe SOS**: 질문자가 크레딧을 걸고 질문을 올리면, 답변자가 해결책을 제시하고 크레딧을 획득합니다.
- **수수료 모델**: 사용자 간 크레딧 거래 시 일정 비율(예: 10-20%)의 운영 수수료를 시스템이 취득하여 운영비에 충당합니다.
- **Collaborative Growth**: 뛰어난 결과물을 만든 사용자는 자신의 '커스텀 레시피'를 유료 크레딧으로 판매할 수 있습니다.

---

## 4. 'The vibers'만의 핵심 경쟁력 (Moat)

1. **No-Storage Policy**: 사용자의 코드를 우리 서버에 저장하지 않습니다. 오직 사용자의 GitHub으로 직접 전송합니다. (보안 및 신뢰도 극대화)
2. **Knowledge as Service**: 단순한 코딩 툴이 아닙니다. 레오님이 수집한 **'가장 잘 팔리는 AI 디자인 레시피'**를 담고 있는 유일한 플랫폼입니다.
3. **Voice-First Experience**: 텍스트보다 직관적인 **'음성 지휘'** 경험에 모든 UX를 집중합니다.

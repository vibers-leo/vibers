# NCP Object Storage 마이그레이션 + arthyun 통합 완성
- 날짜: 2026-03-30
- 에이전트: [TCC]

## 세션 개요
Firebase Storage → NCP Object Storage(wero-bucket) 전면 마이그레이션 완료.
arthyun/art-way 앱을 artpage 멀티테넌트 아키텍처로 통합하는 기반 작업 완성.

## 주요 논의

### 1. NCP Object Storage 세팅
- 버킷명: `wero-bucket` / 리전: kr-standard
- Access Key: `ncp_iam_****` (Vercel 환경변수 NCP_ACCESS_KEY에 저장)
- 퍼블릭 URL: `https://wero-bucket.kr.object.ncloudstorage.com/{key}`
- **Vercel에 NCP 키를 직접 넣지 않는 전략 채택**: VM의 Docker 컨테이너(vibers-upload, port 8091)가 NCP로 전송, Vercel 앱은 VM 프록시만 호출

### 2. VM Docker 컨테이너 추가 (vibers-upload)
- `/opt/vibers-upload/server.js` — Node.js/Express, port 8091
- `POST /upload` → multer로 파일 수신 → NCP Object Storage PutObject
- 응답: `{ url: "https://wero-bucket.kr.object.ncloudstorage.com/..." }`
- 기존 vibers-storage(port 8090)는 Nginx 정적 파일 서빙 전용 — 업로드 기능 없음

### 3. artpage API route 변경
- `src/app/api/upload/route.ts`: Vercel → VM:8091 프록시로 단순화
- `src/lib/ncp-storage.ts`: S3Client 직접 업로드 유틸 (서버 컴포넌트용)
- `next.config.ts`: `*.kr.object.ncloudstorage.com` 도메인 추가

### 4. Firestore → PostgreSQL 마이그레이션 (arthyun)
- VM `/opt/migrate/firestore-to-pg.js` 실행
- Firebase 서비스 계정: `arthyun-5b255-firebase-adminsdk-...json` (마이그레이션 후 VM에서 삭제)
- 결과: **총 188개 레코드 삽입**
  - portfolio: 184개 (이미지 Firebase Storage → NCP Object Storage 이전)
  - media_releases: 2개 (이미지 이전)
  - site_settings: 1개
  - inquiries: 1개
  - exhibitions: 0개 (Firestore에 없었음)

### 5. 코드 수정 내용
- `arthyun/page.tsx`: Firestore SDK 제거 → Prisma `prisma.portfolio.findMany`
- `art-way/page.tsx`: `main_banner.youtube_url` → `site_settings.youtube_url` (스키마 수정)
- `[slug]/page.tsx`: 동적 라우트도 Prisma 기반으로 전환

## 기술적 상세

### NCP S3 클라이언트 설정
```typescript
const s3 = new S3Client({
  region: "kr-standard",
  endpoint: "https://kr.object.ncloudstorage.com",
  credentials: { accessKeyId: ..., secretAccessKey: ... },
  forcePathStyle: false,  // Virtual-hosted style (버킷.엔드포인트 형식)
})
```

### 발견한 스키마 이슈
- `prisma.Portfolio` 존재 안 함 → `prisma.portfolio` (소문자)
- Portfolio 모델에 `is_visible` 없음 → `is_featured`만 존재
- `main_banner.youtube_url` 없음 → `site_settings.youtube_url`에 있음

## 향후 계획
1. **Vercel 대시보드**: artpage 프로젝트에 `arthyun.co.kr` 도메인 추가 (수동)
2. **art-way Firebase 마이그레이션**: art-way는 별도 Firebase 프로젝트 — 서비스 계정 키 필요 시 진행
3. **arthyun.co.kr 도메인 DNS**: arthyun Vercel 프로젝트 → artpage Vercel 프로젝트로 이전
4. **VM 보안**: 서비스 계정 키 `/opt/migrate/service-account.json` 삭제 완료 ✅

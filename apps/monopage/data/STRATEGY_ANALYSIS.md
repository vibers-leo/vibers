# ArtPage (북촌 아트 스페이스) 전략 진단 리포트

> PM Skills 프레임워크 기반 전략 분석 | 2026-03-26

---

## 1. 프로젝트 개요 및 현재 상태
- **역할**: 북촌 아트 스페이스 홈페이지 — 갤러리/전시 관리 웹사이트
- **개발 완성도**: 70% (기본 구조 완성, 콘텐츠 채우기 단계)
- **에코시스템 내 위치**: Vibers 클라이언트 사이트 — 실제 갤러리 운영 사용사례

## 2. 핵심 가치 & 포지셔닝
- **클라이언트 가치**: 북촌 지역 아트 스페이스의 디지털 전시 허브
- **기술 스택**: Supabase SSR + Tiptap 에디터 — 경량 솔루션 증명
- **AI Recipe 활용**: 전시 이미지 보강 및 갤러리 홍보 비주얼 자동화
- **포트폴리오 가치**: 소규모 갤러리/전시 공간 웹사이트 구축 템플릿

## 3. 주요 리스크 & 과제
1. **콘텐츠 의존성**: 갤러리 운영자의 콘텐츠 업로드 빈도에 따른 활성도 차이
2. **SEO 최적화 미흡**: 전시 정보 검색 가시성 개선 필요
3. **모바일 경험**: 갤러리 방문객의 모바일 브라우징 체험 고도화

## 4. 권장 액션
- **단기(1개월)**: 전시 일정 관리 기능 강화, 모바일 UI/UX 개선
- **중기(3개월)**: SEO 최적화, SNS 연동(Instagram 갤러리 자동 동기화) 구현

## 핵심 한 줄 요약
소규모 갤러리/전시 공간의 웹 전환 레퍼런스 — Supabase SSR 기반 경량 솔루션으로 오프라인 공간의 온라인 허브 역할.

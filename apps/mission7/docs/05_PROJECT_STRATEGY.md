## 2. 미션7로 관리하는 7대 프로젝트 전략

미션7의 "프로젝트 미션" 기능으로 각 프로젝트의 완성도를 추적한다.
주간 AI 리뷰에서 진행률을 분석하고, 다음 주 미션을 자동 추천한다.

### 프로젝트 1: 나노바나나2 가이드북
| 항목 | 내용 |
|------|------|
| 현재 진행률 | 약 25% (기획안 완료, 원고 초안 시작) |
| 미션 타입 | 프로젝트 (0→100%) |
| 주간 목표 | 매주 2챕터 완성 (주 3회 미션) |
| 마일스톤 | 4/15 초안 완료 → 4/22 교정 → 4/30 PDF 출시 |
| Beachhead | AI 이미지 생성에 관심 있는 크리에이터 |
| 수익 모델 | 전자책 판매 (크몽/탈잉/자체) |

**미션7 등록 예시:**
- 일일 반복: "나노바나나2 1시간 집필"
- 프로젝트: "나노바나나2 가이드북" (진행률 추적)
- D-day: "초안 완료" (4/15)

### 프로젝트 2: 바이브코딩 전자책
| 항목 | 내용 |
|------|------|
| 현재 진행률 | 0% (기획 전) |
| 미션 타입 | 프로젝트 (0→100%) |
| 주간 목표 | 나노바나나2 완료 후 착수 (5월~) |
| 콘텐츠 소스 | Claude 대화 기록, 개발 세션 로그, 프로젝트 경험 |
| Beachhead | 비개발자 중 AI 코딩에 관심 있는 사람들 |
| 차별점 | 실제 프로젝트(미션7, 팬이지 등)를 만든 실전 기록 |

**전략:** 나노바나나2 가이드북 출시 경험을 바탕으로 동일 파이프라인 적용.
미션7 앱 자체가 바이브코딩의 케이스 스터디가 됨.

### 프로젝트 3: 계발자들 유튜브 채널
| 항목 | 내용 |
|------|------|
| 현재 진행률 | 0% |
| 마감 | 4/3 (D-6) |
| 미션 타입 | D-day |
| 콘텐츠 | "하루 1시간 치열하게 개발하는 영상" |
| 배포 | 유튜브 + 스레드 |

**주간 미션 분해:**
- 3/29(일): 채널 이름/컨셉 확정, 프로필 기획
- 3/30(월): 유튜브 채널 개설 + 프로필 설정
- 3/31(화): 첫 영상 기획 (미션7 개발 과정 촬영)
- 4/1(수): 촬영 (코딩 1시간 타임랩스)
- 4/2(목): 편집 + 업로드
- 4/3(금): 스레드 공유 + 두번째 영상 기획

**Growth Loop:** 유튜브 영상 → 스레드 공유 → 바이브코딩 관심층 유입 → 계발자들 인지도 → CTO서비스/앱외주 리드

### 프로젝트 4: 계발자들 에이전시 플랫폼
| 항목 | 내용 |
|------|------|
| 현재 | vibers.co.kr 도메인 확보, 사업계획서 작성 완료 |
| 포지셔닝 | 기술력 있는 앱 외주 에이전시 |
| 스택 | Next.js (vibers 앱) + Rails API |
| 우선순위 | 유튜브/콘텐츠로 리드 확보 후 강화 (5~6월) |

**미션7 역할:** vibers.co.kr 개선 관련 미션을 프로젝트 타입으로 등록하여 진행률 추적.

### 프로젝트 5: 팬이지 정기결제

## 7. 성공 지표 (pm.md North Star)

### North Star Metric
**월간 퍼펙트 일수 (Monthly Perfect Days)**

LEO가 한 달에 7/7 퍼펙트를 달성하는 날의 수.
이 숫자가 올라가면 모든 프로젝트가 전진한다.

### Input Metrics
| 지표 | 목표 | 측정 |
|------|------|------|
| 일일 미션 세팅률 | 90%+ | 매일 아침 7미션 세팅하는 비율 |
| 주간 달성률 | 80%+ | 주간 완료 미션 / 주간 전체 미션 |
| 프로젝트 진행률 증가 | 주 5%+ | 프로젝트 미션 진행률 주간 증가분 |
| 수익 미션 비율 | 30%+ | 수익 관련 미션 / 전체 미션 |

## 8. 미션7 CLAUDE.md 프롬프트 (개발용)

개발 시작 시 `nextjs/apps/mission7-app/CLAUDE.md`에 넣을 내용:

```markdown
# Mission7 — Expo 앱

## 개요
게임하듯 즐겁게 매일 7개 미션을 달성하는 목표달성 앱.
"하루 7미션, 반드시 성공합니다."

## 스택
- Expo (React Native) — SDK 52+
- TypeScript strict
- React Navigation
- Firebase Auth (Google/Apple)
- Rails API (mission7.vibers.co.kr)
- Reanimated + Haptics + Lottie

## 핵심 규칙
1. 미션은 최대 7개 (slot_index 0~6)
2. 모든 애니메이션은 Reanimated (60fps)
3. 완료 시 반드시 햅틱 피드백
4. 7/7 달성 시 퍼펙트 이펙트 필수
5. 다크 테마 기본 (Vantablack Luxe)
6. 한글 우선, Pretendard 폰트
7. gstack.md + pm.md 빌더 원칙 준수

## 참고 문서
- docs/01_USER_WORKFLOW.md — 유저 워크플로우
- docs/02_API_SPEC.md — API 명세
- docs/03_DB_SCHEMA.md — DB 스키마
- docs/04_FEATURE_SPEC.md — 기능명세
- docs/05_PROJECT_STRATEGY.md — 전략
- docs/design-preview.html — 디자인 프리뷰

## 개발 명령어
bun run dev --filter=mission7-app
```

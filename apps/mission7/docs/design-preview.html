RPG Quest UI — 디자인 노트

컨셉: SF RPG 미션 브리핑. 매일 아침 퀘스트가 "도착"하고, 클리어하면 XP를 얻고, 7개 전부 클리어하면 "ALL CLEAR" + 레벨업. 생산성 도구가 아닌 게임을 하는 느낌.

비주얼 언어: Vantablack + 홀로그래픽 시머. 배경에 떠다니는 ambient orb, 카드에 rune-border (그라디언트 보더), 완료 시 글로우 이펙트. 중세 판타지가 아닌 "데스티니/사이버펑크 HUD" 무드.

게임 용어 매핑: 미션→퀘스트, 통계→전적, 완료→CLEARED, 7/7→ALL CLEAR, 스트릭→연속 ALL CLEAR, 프로젝트→퀘스트 체인(CHAIN), 중요 미션→EPIC, XP 시스템으로 레벨업.

퀘스트 등급: 일반(기본), CHAIN(장기 프로젝트, 보라색), EPIC(D-day/중요, 금색), LEGENDARY(특별 업적, 미래).

모션: 퀘스트 도착 시 우측에서 슬라이드인(questArrive), 슬롯 활성화 시 스케일 바운스+글로우(slotActivate), ALL CLEAR 시 파티클 상승+홀로 텍스트+랭크업.

컬러: void(#06060e) 베이스, accent(Blue), xp(Purple), gold(Amber), quest(Emerald), hp(Red). 카테고리 구분과 게임 의미를 동시에 전달.

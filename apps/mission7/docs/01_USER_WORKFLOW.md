## 1. 앱 최초 진입 (온보딩)

### 1-1. 스플래시 → 온보딩
```
스플래시 (Mission7 로고 + 보상 이펙트 미리보기)
  ↓
온보딩 3스텝:
  ① "매일 7개의 미션을 달성하세요" — 컨셉 설명
  ② "스스로 정한 미션이 가장 강력합니다" — 자기주도 강조
  ③ "지금, 첫 미션을 만들어볼까요?" — AI 대화 시작
  ↓
Firebase Auth 로그인 (Google / Apple / 이메일)
  ↓
AI 코칭 첫 대화 (선택사항):
  "안녕하세요! 요즘 가장 이루고 싶은 목표가 뭔가요?"
  → 사용자 답변 기반으로 미션 추천 3~5개 생성
  → 사용자가 선택/수정하여 첫 미션 세팅
```

### 1-2. 미션 프리셋 (빠른 시작)
AI 대화를 건너뛰고 싶은 사용자를 위해 카테고리별 프리셋 제공:
- 건강/운동: "30분 걷기", "물 8잔 마시기", "스트레칭 10분"
- 자기계발: "책 30분 읽기", "영어 단어 20개", "일기 쓰기"
- 생산성: "1시간 딥워크", "이메일 제로", "오늘 할 일 정리"
- 창작/개발: "코딩 1시간", "블로그 글 쓰기", "영상 편집 30분"

│                                 │

│                                 │

### 2-2. 미션 추가 방법 (4가지)
| 방법 | 설명 | UX |
|------|------|-----|
| 직접 입력 | 텍스트로 미션 작성 | 텍스트 입력 + 미션 타입 선택 |
| 반복 미션 | 어제/매일 반복 미션 자동 제안 | 탭하면 바로 추가 |
| AI 추천 | AI가 맥락 기반 추천 | 카드 스와이프로 선택/스킵 |
| 보관함 | 미리 만들어둔 미션 목록 | 체크박스 다중 선택 |

### 2-3. 미션 타입 설정 (추가 시 선택)
```
미션 타입 선택:
  ○ 일일 — 오늘만 (기본값)
  ○ 반복 — 매일 / 주 N회 / 월 N회
  ○ D-day — 마감일 있는 목표
  ○ 프로젝트 — 완성도 추적 (0~100%)
```

## 4. AI 코칭 플로우

### 4-1. AI와 대화하는 시점
| 시점 | 트리거 | AI 역할 |
|------|--------|---------|
| 미션 설정 시 | "+ AI와 미션 만들기" 버튼 | 목표 수립 코칭 |
| 미션 진행 중 | AI 탭 진입 | 격려, 우선순위 제안 |
| 미션 미완료 시 | 자정 전 알림 | 리프레이밍, 내일 계획 |
| 주간 리뷰 | 일요일 저녁 알림 | 주간 성과 분석, 다음주 추천 |

### 4-2. AI 비용 최적화 구조
```
사용자 입력
  ↓
[라우터 — 로컬 판단]
  ├─ 단순 격려/리마인더 → 템플릿 기반 (API 호출 없음)
  ├─ 미션 추천 → Gemini Flash (저비용, 빠른 응답)
  └─ 전략 대화/목표 수립 → Claude API (고품질, 세션당 1~2회)
```

### 4-3. AI 대화 예시
```
사용자: "이번 주 목표를 세우고 싶어"
AI: "지난주에 달성률이 높았던 미션을 보면, '코딩 1시간'과 '걷기'는
     매일 꾸준히 하셨네요. 이번 주에 새로 도전해보고 싶은 분야가 있나요?"
사용자: "유튜브 채널을 개설하고 싶어"
AI: "좋은 목표네요! 이번 주 안에 달성할 수 있게 쪼개볼까요?

     월: 채널 이름/컨셉 정하기 (30분)
     화: 채널 개설 + 프로필 설정 (1시간)
     수: 첫 영상 기획/스크립트 (1시간)
     목: 촬영 (1시간)
     금: 편집 + 업로드 (1시간)

     이렇게 5일 미션으로 나누면 어떨까요?"
```

## 5. 통계/대시보드 플로우

### 5-1. 일간 리포트
- 완료 미션 수 / 7
- 가장 빨리 완료한 미션
- 가장 오래 걸린 미션
- AI 한줄 코멘트

### 5-2. 주간 리포트
- 주간 달성률 그래프 (월~일)
- 퍼펙트 일수
- 카테고리별 분포 (건강/생산성/창작 등)
- 연속 스트릭 현황

### 5-3. 월간/장기 리포트
- 월간 히트맵 (GitHub 잔디밭 스타일)
- 프로젝트 미션 완성도 추이
- D-day 미션 진행률
- AI 종합 피드백

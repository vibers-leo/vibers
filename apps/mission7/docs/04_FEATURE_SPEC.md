# 미션7 (Mission7) — 세부 기능명세서

> 작성일: 2026-03-28
> 버전: MVP (v0.1)
> 플랫폼: Expo (React Native) + Next.js (웹 대시보드)

---

## MVP 스코프 정의

### 포함 (v0.1)
- 미션 CRUD (생성/조회/수정/삭제)
- 7슬롯 시스템
- 미션 완료/스킵 처리
- 프로젝트 미션 진행률
- 보상 이펙트 (파티클, 햅틱, 사운드)
- 기본 통계 (일간/주간)
- 푸시 알림 리마인더
- Firebase Auth (Google/Apple)

### 제외 (v0.2+)
- AI 코칭 대화
- AI 미션 추천
- 소셜/공유 기능
- 친구/그룹 챌린지
- 웹 대시보드
- 카카오 미니앱

## F2. 메인 화면 (오늘의 미션)

### F2-1. 7슬롯 디스플레이
| 항목 | 상세 |
|------|------|
| 레이아웃 | 상단에 7개 원형 인디케이터 (가로 배열) |
| 빈 슬롯 | 회색 원 (outline) |
| 완료 슬롯 | 카테고리 색상으로 채워진 원 |
| 진행중 슬롯 | 펄스 애니메이션 |
| 카운터 | "오늘의 미션 3/7" 텍스트 |

### F2-2. 미션 리스트
| 항목 | 상세 |
|------|------|
| 레이아웃 | 세로 스크롤 카드 리스트 |
| 카드 구성 | 카테고리 색상 바 (좌측) + 제목 + 미션타입 뱃지 + 완료 버튼 |
| 정렬 | slot_index 순서 (드래그로 변경 가능) |
| 빈 상태 | "미션을 추가해주세요" + 추가 버튼 |
| 날짜 네비게이션 | 상단 좌우 스와이프로 어제/내일 조회 (읽기 전용) |

### F2-3. 미션 추가 (Bottom Sheet)
| 항목 | 상세 |
|------|------|
| 트리거 | FAB 버튼 또는 "+" 카드 탭 |
| 입력 필드 | 미션 제목 (텍스트), 카테고리 (선택), 미션타입 (선택) |
| 카테고리 선택 | 6개 아이콘 버튼 (가로 스크롤) |
| 미션타입 | 일일(기본) / 반복 / D-day / 프로젝트 |
| 반복 설정 | 타입이 반복일 때: 매일 / 주 N회 / 월 N회 |
| D-day 설정 | 타입이 D-day일 때: 날짜 피커 |
| 프로젝트 설정 | 타입이 프로젝트일 때: 현재 진행률 슬라이더 |
| 보관함 저장 | 토글: "보관함에 저장" (템플릿 생성) |
| 제한 | 7개 초과 시 추가 불가, 안내 메시지 표시 |

### F2-4. 빠른 추가 (추천 섹션)
| 항목 | 상세 |
|------|------|
| 어제 완료 | 어제 completed인 미션 중 오늘 없는 것 표시 |
| 고정 미션 | is_pinned=true 템플릿 중 오늘 없는 것 표시 |
| 보관함 | "보관함에서 선택" 버튼 → 템플릿 리스트 |
| 탭 동작 | 추천 미션 탭 시 바로 오늘에 추가 (슬롯 자동 배정) |

## F3. 미션 수행

### F3-1. 완료 처리 (일반 미션)
| 항목 | 상세 |
|------|------|
| 트리거 | 카드 우측 원형 버튼 탭 |
| 애니메이션 | 체크마크 드로잉 (Lottie 또는 Reanimated) |
| 햅틱 | 중간 강도 진동 (Haptics.impactAsync) |
| 사운드 | 짧은 "팅" 효과음 (선택적, 설정에서 끄기 가능) |
| 상태 변경 | status: pending → completed, completed_at 기록 |
| 슬롯 업데이트 | 상단 인디케이터 즉시 반영 |
| 되돌리기 | 완료 카드 탭 시 "되돌리기" 확인 후 pending으로 복귀 |

### F3-2. 진행률 업데이트 (프로젝트 미션)
| 항목 | 상세 |
|------|------|
| UI | 카드 내 슬라이더 (0~100%) |
| 인터랙션 | 손가락 드래그로 진행률 조절 |
| 애니메이션 | 게이지 채워지는 물결 이펙트 |
| 100% 도달 | 자동으로 completed 처리 + 특별 이펙트 |
| 템플릿 동기화 | 프로젝트 진행률은 template.current_progress에도 반영 |

### F3-3. 미션 스킵
| 항목 | 상세 |
|------|------|
| 트리거 | 카드 길게 누르기 → 컨텍스트 메뉴 → "오늘 스킵" |
| 사유 입력 | 선택적 (기분 안 좋음 / 일정 변경 / 기타) |
| 상태 | status: skipped |
| 슬롯 | 회색으로 변경, 카운터에서 제외 |

### F3-4. 미션 편집/삭제
| 항목 | 상세 |
|------|------|
| 트리거 | 카드 길게 누르기 → 컨텍스트 메뉴 |
| 편집 | 제목, 카테고리, 타입 변경 가능 |
| 삭제 | 오늘 슬롯에서 제거 (템플릿은 유지) |
| 순서 변경 | 카드 드래그앤드롭으로 slot_index 변경 |
